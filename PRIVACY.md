# OIC Activity Stream Enhanced Viewer — Privacy Policy

The OIC Activity Stream Enhanced Viewer Chrome extension does not collect, store on remote servers, or transmit any user data to third parties.

What stays on your device only:
- UI preferences (theme, auto-detect button toggle) saved in chrome.storage.local on your computer.

Network requests:
- Same-origin GET requests to your authenticated Oracle Integration Cloud instance (under *.oraclecloud.com) — to fetch the activity-stream JSON and any large payloads on demand. These requests are authenticated by your existing OIC session cookies; no separate credentials are stored or transmitted.

The extension contacts no third-party servers, analytics services, or telemetry endpoints.

For questions, please open an issue at:
https://github.com/marat-ak/oic-activity-viewer-extension/issues
